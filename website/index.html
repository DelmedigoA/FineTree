How it works

From raw PDF to benchmark-ready dataset

Three stages that cover the complete annotation and training lifecycle.

01

Annotate

Open any financial PDF in the desktop annotator. Review page-by-page, assign page types, and extract structured facts with AI-assisted suggestions from Gemini or Qwen.

02

Build & Train

Export approved annotations as training JSONL. Push datasets to Hugging Face automatically, then fine-tune models locally or on RunPod with a single command.

03

Benchmark

Score model submissions against ground-truth annotations through the web UI or headless runner. Track accuracy across fact types and compare runs in a persistent leaderboard.

Features

Built for financial document intelligence

Every tool in the pipeline is designed specifically for the challenges of structured data extraction from financial reports.

Desktop Annotation Studio

PyQt5-based annotator with PDF import, page-level review, and side-by-side JSON editing. Keyboard-driven workflows for high-throughput annotation sessions.

AI-Powered Extraction

Gemini and Qwen VLM integration for automatic fact extraction from page images. Supports bbox-only mode, full extraction, and streaming generation for real-time review.

Schema Validation

Pydantic-enforced canonical schemas with equation integrity checks ensure every annotation is consistent and export-ready.

HF Dataset Export

Build training JSONL from approved annotations and push directly to Hugging Face Hub with a single command.

Multiple Serve Modes

FastAPI, Gradio, RunPod serverless, and pod entrypoints — deploy where it makes sense for your workflow.
